StelliesLive is a student-focused event discovery app based in South Africa. Our goal is to help users discover the best events, student nights, and live music in Stellenbosch.

2. What Personal Information We Collect

a. For General Users (No Login Required)

b. For Admin Users

  • Email address (via Firebase Authentication)
  • Event data submissions

6. Data Sharing and Third Parties

We use third-party services:

  • Firebase Authentication (Google LLC)
  • Firebase Firestore
  • Firebase Analytics
  • Firebase Functions
  • AdMob by Google (only on mobile)
  • Google Maps API

These services may collect anonymized analytics and device identifiers. All data transmitted is encrypted in transit. We do not sell or trade user data.

7. Data Security

  • All communication is encrypted via HTTPS/TLS
  • Firebase Authentication controls admin access
  • Firestore databases are access-controlled
